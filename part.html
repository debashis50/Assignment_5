Question-3: What is event delegation in the DOM?

Event delegation allows a parent element to handle events for its children using event bubbling. It improves performance, works for dynamic elements, and simplifies code.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event on a child element travels up the DOM tree to its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?
